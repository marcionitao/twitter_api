# Twitter Clone API

## Description

The Twitter Clone API is a REST API that allows users to create, read, update, and delete tweets. It also allows users to follow other users and get their timeline of tweets.

## Technologies Used

- Node.js + types
- Express.js + types
- TypeScript
- Prisma
- PostgreSQL
- Zod
- JWT + types
- slug + types
- bcrypt-ts

- helmet
